## üèÅ Getting Started

Follow the steps below to get this project up and running on your local machine.

### 1. Prerequisites

Ensure you have the following software installed on your computer:

- [Node.js](https://nodejs.org/) (Recommendation: v22.x or later)

### 2. Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/mhndkptr/Learniverse-FE.git
    cd Learniverse-FE
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    # pnpm install
    # or
    # yarn install
    ```

3.  **Set up Environment Variables:**
    Copy the `.env.example` file to a new file named `.env.local` and fill in the necessary variables.
    ```bash
    cp .env.example .env.local
    ```

### 3. Running the Project

**Running the Development Server:**
To run the application in development mode with hot-reloading:

```bash
npm run dev
# or
# pnpm dev
```

Open http://localhost:3000 in your browser to see the result.
